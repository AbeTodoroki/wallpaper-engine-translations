[h3]将新壁纸添加到 Android[/h3]
从技术和法律的角度来看，从移动设备直接连接到创意工坊是不可能的。 但是，我们构建了一个便利的系统，可让 Wallpaper Engine 的所有者从 Windows 上轻松地将壁纸传输到 Android 设备：

[b]第 1 步：[/b]在 Windows 上的 Wallpaper Engine 中单击新的[b]移动设备[/b]按钮，然后选择[b]连接新设备[/b]。 Wallpaper Engine 将显示 4 位数 PIN，您可以立即使用该 PIN 连接到您的移动设备。

[b]第 2 步：[/b]在您的 Android 设备上打开 Wallpaper Engine，然后单击右下角的[b]添加[/b]按钮。 选择[b]与计算机配对[/b]，此时您的计算机应该会出现在设备列表中。 选择您的计算机，然后输入第 1 步中的 4 位数 PIN。

[b]第 3 步：[/b]您的设备现在应该已连接。 右键单击任何兼容的壁纸，并单击[b]发送到移动设备[/b]，然后从列表中选择您的移动设备。 您的壁纸现在将会传输。

[img]{STEAM_CLAN_IMAGE}/26312799/0fc8bc609bd7d959fa87fe7f90e3f1612ba67cd1.gif[/img]

[b]如果您无法连接到 Android：[/b]右键单击任何壁纸并选择[b]发送到移动设备[/b]，然后选择[b]导出 .mpkg[/b]。 然后，您可以通过数据线或其他方式将导出的移动包文件传输到您的移动设备。

[list][*] [url=https://help.wallpaperengine.io/mobile/pairing.html]单击此处阅读我们的 Android 连接指南。[/url][/list]
[i]提示：您也可以通过按住 CTRL 键多选文件夹或多个壁纸来同时传输它们。 在[b]已安装[/b]选项卡中按[b]CTRL + A[/b]选择所有已安装的壁纸，以便同时传输这些壁纸。[/i]

[h2]工作日播放列表[/h2]
随着时间的推移，我们收到的一个普遍请求是每个工作日配置一张壁纸的方法。 在本次更新中，我们引入了新的工作日播放列表。 只需在 Wallpaper Engine 的[b]已安装[/b]选项卡中将最多七张壁纸添加到播放列表，然后在播放列表部分单击[b]配置[/b]并从中选择[b]星期几[/b]选项。 您还可以在其中找到现有的播放列表选项，例如[b]当日时间[/b]播放列表类型。

[h2]Windows 11[/h2]
Wallpaper Engine 完全支持 Windows 11，在本次更新中，我们还增加了对新的圆角用户界面设计的支持。 如果您正在升级到 Windows 11 并在使用 Wallpaper Engine 时遇到任何问题，请确保完全重新安装最新的显卡驱动程序。 大部分向我们提出问题的 Windows 11 用户都使用了过时的显卡驱动程序。

如果您使用的是具有透明度的 Windows 11 主题，您会注意到 Windows 11 会忽略 Wallpaper Engine。 要解决此问题，只需在 Wallpaper Engine 设置中启用新的[b]覆盖壁纸[/b]选项。 这将使您的 Windows 壁纸与 Wallpaper Engine 壁纸保持同步，并修复 Windows 11 上的任何透明度问题。

[h2]有关 Android 应用程序的常见问题[/h2]

[h3]Android 应用程序的费用是多少？[/h3]
Android 应用程序完全免费，没有隐藏费用，没有应用程序内购买，也没有广告。 它也可以作为独立的应用程序使用，而无需在计算机上安装 Wallpaper Engine，并从您的移动设备导入兼容的壁纸文件。

[h3]可以直接从移动应用程序访问创意工坊吗？[/h3]
不可以，由于技术和法律限制，无法从移动设备访问创意工坊。 我们构建了一种便利的方式来将您现有的壁纸合集从计算机传输到 Android。 我们竭尽全力让这个过程尽可能简便，在大多数情况下，您只需要确保移动设备和计算机连接到同一网络。 [url=https://help.wallpaperengine.io/mobile/pairing.html]单击此处了解更多。[/url]

此外，您还可以直接从移动设备上存储的文件中导入任何兼容的壁纸文件。 如果您只是想寻找一种方法将 Android 手机或平板电脑上的任何视频文件转换为手机壁纸，或者您想在播放列表中使用多个静态壁纸，那么 Android 版 Wallpaper Engine 也能有效发挥作用。

[h3]Wallpaper Engine 会耗尽我设备的电池电量吗？[/h3]
不会，我们已经广泛优化了应用程序，大多数现代设备在日常使用中不会注意到对电池寿命的负面影响。

我们已经在我们的私人设备上使用该应用程序达数月之久，并且平均每天都在使用，我们还没有通过 Wallpaper Engine 看到任何有意义的电池使用情况。 当另一个应用程序在您的设备上全屏显示以及您的设备关闭时，该应用程序会自动暂停。 省电模式会自动暂停您的动态壁纸，以进一步减少其对电池寿命的影响。 除此之外，您还可以通过在应用程序设置中调整 FPS 限制来调整性能影响。

[h3]壁纸也会像在 Windows 上一样播放声音吗？[/h3]
不会，所有壁纸在 Android 上都是静音的。 但是，只要您授予应用程序录制设备音频的权限，音频可视化工具就可以工作。

[h3]我能在 Android 锁屏上设置壁纸吗？[/h3]
这取决于您的设备。 一些 Android 制造商不允许在锁屏上使用动态壁纸，而另一些制造商则允许。 如果没有为您提供设置锁屏壁纸的选项，则您的设备制造商不允许这样做。

[h3]适用于 iPhone 和 iPad 的 iOS 版 Wallpaper Engine 呢？[/h3]
不幸的是，Apple 的 iOS 不像 Android 一样支持真正的动态壁纸。 因此无法在 iOS 上创建任何类似于 Wallpaper Engine 的内容。 如果 Apple 改变了主意，我们可能还会跟进 iOS 版本。 [url=https://help.wallpaperengine.io/functionality/linuxmacos.html]单击此处了解我们对 Linux、Mac OS 和 iOS 的详尽说明。[/url]
